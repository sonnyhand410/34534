# 🔫 Shooting Game

A simple 2D shooting game made with Python and Pygame.

## 🎮 How to Play

- Move left/right using the arrow keys
- Press SPACE to shoot bullets
- Shoot the enemies before they reach the bottom
- Each enemy hit increases your score!
